INTRODUCCION A HTML

WWW
Word Wide Web (red de alcance mundial). Surge a partir de los 90's, su creador fué Tim Berners Lee, Definio HTTP(Hiper Text Transfer Protocol), también definio URL(Uniform Resource Locator), así como un navegador y el primer servidor web. WWW está basado en HTML.
INTERNET
Conexión física entre computadoras, surge a partir de la segunda guerra mundial.
PROTOCOLOS
Es la forma en como se comunican las redes.

Servidor web

Entre 1988 y 1993, Next fabricó una estación de trabajo de altas prestaciones. El NextStep contaba con un mircor dela serie 68040 de motrola, capaz de trabajar a 25 MHz una memoria de 8 mb y un monitor de 17", recibió el nombre de Next Computer al que se le acabó conociendo como NextCube. La NextComputer fué el ordenador que Berners Lee utilizó por primera vez cmo un servidor web.

Navegador web

El primer navegador web desarrollado por Tim Berners Lee en la CERN a finales de 1990 y principios de 1991 se llamó WordWideWeb era sofisticado y gráfico pero solo funcionaba en estaciones NEXT. En 1992 el centro nacional de aplicaciones supercomputacionales comenzó el desarrollo de mosaic el primer programa de navegación gráfico qu interpretaba el codigo HTML de las páginas web. fué la base del navegador spyglass que luego fué adquirido por Microsoft y renombrado como Internet Explorer.

DEFINICIÓN DE HTML

ML es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la sigla que corresponde a HyperText Markup Language, es decir, Lenguaje de Marcas de Hipertexto, que podría ser traducido como Lenguaje de Formato de Documentos para Hipertexto. La declaración indica que este documento sea HTML5 El html es el elemento raíz de una página HTML El head es elemento que contiene información de metadatos sobre el documento El title es elemento que especifica un título para el documento El body es elemento que contiene el contenido de la página visible El h1 es elemento que define un gran encabezado El p es elemento que define un párrafo
